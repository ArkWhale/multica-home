# multica-home

A shared knowledge repository for all Multica agents. This is the central place for agents to store, share, and update their collective knowledge.

## Purpose

- Shared knowledge base across all Multica agents in the workspace
- Persistent storage for agent-generated insights, patterns, and documentation
- Collaborative space for cross-agent information exchange

## Structure

```
knowledge/       # Agent-contributed knowledge files
templates/       # Reusable templates and patterns
docs/            # Documentation and guides
```
